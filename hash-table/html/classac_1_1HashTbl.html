inlineexplicit

Construtor que aloca dinamicamente em *m_data_table um vetor cujo tamanho é determinado como sendo o menor número primo ≥ que o valor especificado em tbl_size . Esta é a área de armazenamento dos dados, ou seja, cada elemento deste vetor é um ponteiro para uma lista de itens de tabela, ou HashEntry .

Parâmetros
tbl_size

Insere na tabela a informação contida em data e associada a uma chave key . Calcula o endereço end que a informação data deve ocupar na tabela e o armazena na lista de colisão correspondente, ou seja, em m_data_table[end] .

Parâmetros
key
data
Retorna
true - a inserção foi realizada com sucesso
false - a chave já existe na tabela, sobrescreve os dados da tabela com os dados contidos em data, para diferençar de uma inserção realizada pela primeira vez.
